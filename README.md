> [vuepress 主题开发协议文档](https://vuepress.vuejs.org/zh/theme/)

## Use

在一个 `vuepress` 项目中使用

将本主题作为依赖安装到项目之中

```bash
npm i vuepress-theme-navi
```

> /docs/.vuepress/config.js

```js
module.exports = {
	// ...
	theme: 'navi',
}
```
